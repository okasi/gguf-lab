Quantization, memory bandwidth, and speculative decoding are the three primary levers controlling token generation speed in local Large Language Models. Quantization reduces the precision of model weights from standard 16-bit or 32-bit floating-point formats to lower-bit integers, such as 4-bit or 8-bit. This compression directly shrinks the model’s footprint, allowing larger architectures to fit into limited GPU VRAM or even system RAM. Crucially, smaller data sizes mean less data needs to be moved, which immediately alleviates pressure on the hardware’s memory bandwidth. Memory bandwidth is often the true bottleneck in local inference because generating tokens requires loading vast amounts of weight data from memory to the processing cores for every single output step. When the model size exceeds available fast memory, the system must fetch data from slower storage, drastically reducing tokens per second. Therefore, maximizing the ratio of compute operations to memory bytes transferred is essential for high throughput. Speculative decoding acts as a sophisticated algorithmic optimization that bypasses this sequential bottleneck.
