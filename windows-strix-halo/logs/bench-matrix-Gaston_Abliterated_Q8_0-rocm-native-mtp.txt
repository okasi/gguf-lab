Quantization reduces the precision of model weights, typically from 16-bit floating point to 8-bit, 4-bit, or even lower integer formats. This compression drastically shrinks the model’s memory foot print, allowing larger architectures to fit entirely within your graphics processing unit’s volatile memory. More importantly, it reduces the amount of data transferred during each forward pass. Since local LLMs are overwhelmingly memory-bound rather than compute-bound, the speed of token generation is directly dictated by memory bandwidth. The GPU must load every weight into high-speed VRAM for each token prediction. Therefore, higher memory bandwidth allows more weights to be read per second, directly increasing throughput. Quantization amplifies this effect by reducing the total bytes moved, effectively multiplying the effective bandwidth available for computation.

Speculative drafting introduces a clever algorithmic optimization to bypass the sequential nature of autore generation.
